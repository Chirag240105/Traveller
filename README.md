# Traveller

Traveller is a premium MERN travel sample built from the original Nomad app. It includes a modern Vite + React frontend, JWT auth, curated hotel listings, booking flow, user profile management, and a sleek premium design.

## Quick start

```bash
cd nomad
npm install
npm install --prefix frontend
npm run dev
```

- Backend server: `http://localhost:5000`
- Frontend app: `http://localhost:5173`

## Features

- Vite + React + TypeScript frontend
- Tailwind CSS styling with premium brand theming
- JWT authentication with email/password login
- Google OAuth sign-in support
- Hotel listing, detail view, booking, and reviews
- Profile page with booking history
- Backend API with Express, MongoDB, and Mongoose

## Available scripts

- `npm run dev` — run backend and frontend concurrently
- `npm run server` — start backend only
- `npm run client` — start frontend only
- `npm run data:import` — import sample hotel data
- `npm run data:destroy` — clear sample data
